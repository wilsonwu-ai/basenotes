# Shopify Product Metafields

For each product, add these custom metafields (Settings → Metafields → Products):

1. `custom.top_notes` - Single line text
2. `custom.heart_notes` - Single line text
3. `custom.base_notes` - Single line text

These will display in the fragrance pyramid on product pages.
