## What these tags enable on the website

- **season:X** — Shows season icons on product cards + powers the seasonal recommendations in cart + enables season filter on collection page
- **origin:X** — Shows "Made in [Country]" below product name on cards
- **event:X** — Enables event filter on collection page

## Tag format

Tags must use the exact `key:value` format with a colon and no spaces around the colon:
- `season:spring` (not `Season: Spring`)
- `origin:france` (not `Origin: France`)
- `event:date-night` (not `Event: Date Night`)
